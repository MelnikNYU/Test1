## Дана задача
Написать программу, которая из имеющегося массива формирует массив 
из элементов менее 3 символов.
Примеры:
* "hello", "world", "2", ":-)" => "2", ":-)"
* "1234", "1567", "-2", "computer science" => "-2"
* "aaaa", "bbbb", "cccc" => []
